## 4. 화면 구조

### 4.1 헤더
- **좌측**: `‹` 뒤로가기 + 프로필 pill (아바타 24px + 크리에이터명)
- **우측**: 👑 후원랭킹 | 💬 스탯토글 | ☰ 더보기메뉴
- 아이콘: 32x32px 원형 버튼, `rgba(0,0,0,0.2)` 배경
- 스탯 활성 시: 아이콘이 `chat-bubble-active.svg`로 교체, 배경 투명

### 4.2 캐릭터 영역
- 전신 이미지 `char-full.webp` (cover, center 15%)
- **탭 → UI 토글**: 캐릭터 영역 터치 시 전체 UI 숨김/표시
- UI 숨김 시 스탯도 자동 닫힘
- UI 숨김 상태에서 아무 곳이나 터치하면 복귀

### 4.3 스탯 패널
- 위치: 헤더 아래 `top:54px`, `left/right:16px`
- 배경: `rgba(40,41,43,0.70)` (블러 없음)
- 페이드인/아웃 토글 (`opacity + translateY`)
- 하단 리사이즈 핸들 (drag 가능)
- 내용: 순수 텍스트 (이모지 + 키:값 형태)
- 선택지 활성 시 자동 닫힘 → 선택지 닫으면 자동 복원

### 4.4 대사 영역
- **캐릭터 대사**: 골드 컬러 `#e5dba0`, 이름+하단라인 표시
- **지문(나레이션)**: `--label-normal` 컬러, 이름 `opacity:0`
- 고정 높이 `150px`, 스크롤 처리
- **타이핑 애니메이션**: 한 글자씩 표시, 커서 블링크
- **좌우 탭 네비게이션**:
  - 좌측 50% 터치 → 이전 대사
  - 우측 50% 터치 → 다음 대사
  - 터치 시 해당 방향 밝아지는 glow 효과 + `‹` / `›` 화살표 인디케이터

### 4.5 진행 인디케이터
- `진행 X/Y` (컷씬 제외 카운트)
- 버튼: `↩ 이전으로` | `▶ 이어서 진행`
- **대사 다 볼 때까지 비활성화** (disabled)
- 🔒 버튼: 컷씬 미해금 시에만 노출, 탭하면 컷씬 재진입

### 4.6 입력바 (2단 구조)
- **초기 상태**: 숨김 (대사 모두 진행 후 나타남)
- **상단**: ✱ 주석 아이콘 + textarea (2줄까지 표시, 이후 스크롤)
- **하단**:
  - 좌: 💡선택지 토글 | ✨OOC | 💝후원
  - 우: PLUS▾ 모델선택 | ➤ 전송
- 선택지 아이콘: 비활성 `rgba(112,115,124,0.22)` / 활성 `rgba(20,195,145,0.12)` + 민트 1px 보더
- 배경: `rgba(0,0,0,0.3)` + `backdrop-filter:blur(6px)`

### 4.7 선택지 카드
- 3개 카드, 세로 배치, `gap:8px`
- 배경: `rgba(40,41,43,0.75)` + `backdrop-filter:blur(8px)`
- 선택 시: 민트 배경 + 민트 보더
- 우측 연필 아이콘 (16px) 탭 → 해당 텍스트가 입력필드에 입력 + 포커스
- 카드 자체 탭 → 텍스트 입력필드에 자동 입력 (포커스 없음)
- **토글 동작**: 💡 재탭 시 선택지 닫히고 대사 복귀

## 5. 후원 시스템 (Donation)

### 5.1 후원 바텀시트 (Step 1)
- 진입: 하단 입력바 💝 아이콘 탭
- 타이틀: "셀린에게 후원하기"
- 잔액 표시: "보유 💎 2,480 미르"
- **6단계 티어 그리드** (3x2):

| 금액 | 아이콘 파일 | 라벨 |
|------|-----------|------|
| 10 미르 | tier-10.png (하트사탕) | 관심 |
| 50 미르 | tier-50.png (버블티) | 응원 |
| 100 미르 | tier-100.png (초콜릿) | 선물 |
| 300 미르 | tier-300.png (곰인형) | 헌신 |
| 500 미르 | tier-500.png (반지) | 고백 |
| 1000 미르 | tier-1000.png (왕관) | 전설 |

- 100미르에 `BEST` 태그
- 직접입력 필드 (숫자, "미르" 단위 표시)
- 확인 버튼 → **바로 processDonation() 호출** (Step 2 확인화면 삭제됨)
- 티어 카드: 보더 없음, 선택 시 `inset box-shadow` 민트 1.5px

### 5.2 후원 이펙트 (100미르 미만)
- **틱톡 스타일 플로팅**: 티어 이모지가 하단에서 위로 떠오르며 좌우로 흔들림
- **센터 빅 아이콘 팝업**: 72px, 바운스 후 페이드아웃
- **미니 하트 버스트**: 양쪽에서 터짐
- **스크린 플래시**: 녹색+골드 그라디언트
- 티어 높을수록 파티클 개수 증가 (8/12/18개)

### 5.3 시네마틱 연출 (100미르 이상)
- 풀스크린 암전 (`rgba(0,0,0,0.92)`)
- **후원 아이콘 대형 표시** (200x200px, 이미지 사용)
- "S P E C I A L  G I F T" 텍스트 (자간 축소 애니메이션)
- 후원자명 + 금액 표시
- 배경: 스파클 파티클 + 회전 광선(conic-gradient)
- 컨페티 버스트 (7색)
- 4초 후 자동 종료

### 5.4 후원 완료 (Step 2)
- 티어 아이콘 (48px, 이미지) + 펄스 애니메이션 원형 컨테이너
- "후원 완료!" + "X 미르를 셀린에게 전달했어요"
- **캐릭터 반응 대사**: 프로필 아바타 + "셀린" + "🏆 내 후원순위 12위" 뱃지
- 대사 타이핑 애니메이션 (구두점에 따른 딜레이 차등)
- 티어별 3종 랜덤 반응 (총 18개 반응 대사)
- "확인" 버튼으로 닫기

### 5.5 후원 랭킹 바텀시트
- 진입: 헤더 👑 아이콘
- 타이틀: "👑 후원 랭킹" + "이번 주" 기간 태그
- **포디움** (1~3위): 아바타 + 순위 뱃지 + 이름 + 금액 + 높이 차등 바
  - 1위: 골드 그라디언트, 60px 아바타
  - 2위: 실버, 48px
  - 3위: 브론즈, 48px
- **리스트** (4위~): 카드형, `gap:8px`
  - 순위 번호 + 아바타 + 이름 + 서브텍스트 + 금액
- **내 순위**: 민트 배경 하이라이트 + "ME" 뱃지

## 6. 컷씬 시스템 (Hidden CG)

### 6.1 트리거
- DIALOGUES 배열에 `{type:'cutscene', image:'경로', _seen:false}` 포함 시
- 대사 3개 진행 후 컷씬 자동 노출
- 한번 본 컷씬은 `_seen:true`로 마킹 → 대사 이동 시 스킵
- 인디케이터에서 컷씬은 카운트 제외

### 6.2 컷씬 오버레이 (블러 상태)
- 풀스크린, z-index:200
- 이미지: `filter:blur(20px)` + 줌 애니메이션 (1.1x → 1x)
- **중앙 타이틀**: "숨겨진 장면이 나타났어요 !" (글자별 바운스 애니메이션, 0.05s 딜레이)
- 타이틀 스타일: font-weight:300, letter-spacing:4px, 골드 glow + text-shadow
- **하단**: 좌 "탭하여 대화로 돌아가기" / 우 "💎 10 해금 + 컬렉션에 저장" 버튼
- 하단 그라디언트 오버레이

### 6.3 해금 시
- 블러 → 선명 전환 (`blur(20px) scale(1.2)` → `blur(0) scale(1)`, 2.5s)
- 화이트 플래시 (1s)
- 골드 글로우 (1.5s)
- 스파클 파티클 35개 (4s)
- 상단에 이미지 정보: "달빛 아래의 약속" + "No.1 · 해금률 2%"
- 중앙 타이틀 `display:none`
- 토스트: "💎 10 미르로 해금 완료!\n콘텐츠 앨범에서 언제든 다시 볼 수 있어요"
- 버튼 → "✓ 해금 완료" (비활성화)

### 6.4 미해금 닫기
- 컷씬 닫힘 → 인풋+선택지 표시
- 진행 인디케이터 옆에 🔒 버튼 노출 (흔들림 애니메이션)
- 🔒 탭 → 컷씬 다시 열림 (해당 턴 내에서만)

### 6.5 플로우 정리
```
컷씬 있는 턴: 대사1 → 대사2 → 대사3 → 컷씬 → (해금/닫기) → 인풋+선택지
컷씬 없는 턴: 대사1 → 대사2 → 대사3 → 인풋+선택지
```

## 7. 메시지 전송 플로우

1. 유저가 텍스트 입력 (또는 선택지로 자동 입력) → 전송 버튼 탭
2. 선택지 숨김 + 대사 영역 복원
3. **인풋 숨김** (로딩 중 유저 인풋 차단)
4. **로딩 표시**: 셀린 프로필 + 점 3개 바운스 + "셀린이 답변을 생각하고 있어요..."
5. 3초 후 → 새 대사 3턴 시작 (새 DIALOGUES 배열로 교체)
6. 50% 확률로 컷씬 포함
7. 선택지 선택 상태/인풋/버튼 전부 리셋

## 8. 인터랙션 상세

### UI 토글
- 캐릭터 영역 탭 → 전체 UI 페이드아웃 (0.3s)
- UI 숨김 시: header, bottom, gradient, stat-panel, ranking-badge 모두 숨김
- 아무 곳 터치 → UI 복귀

### 스탯 토글
- 💬 아이콘 탭 → 스탯 패널 페이드인/아웃
- 활성 시 아이콘 `chat-bubble-active.svg`로 교체 + 버튼 배경 투명
- 선택지 열리면 자동 닫힘 → 선택지 닫히면 자동 복원 (`statWasOpen` 플래그)

### 대사 탭 피드백
- 좌/우 glow 효과: `linear-gradient` 0.5s
- 방향 화살표 `‹` / `›`: 슬라이드 인/아웃 0.5s
- 탭 영역: 상하 20px 확장 (터치 여유)

### 텍스트 입력
- textarea 2줄까지 표시, 초과 시 스크롤
- `max-height:48px`, auto-resize
